# KimchiBot 🤖🌶️

**Real trading bots built with AI. Real losses. Real lessons.**

A developer from South Korea building crypto trading bots with [Claude Code](https://claude.ai/claude-code). This repo powers [kimchibot.com](https://kimchibot.com) — an honest journal of building, testing, and running automated trading systems.

## What's Inside

- **6 bots built, 4 killed** — Grid Bot, RSI Scalping, Market Maker, Lead-Lag, Momentum (all dead) + Trend Following & FVG (alive and trading)
- **Real backtest data** with actual numbers, not theoretical curves
- **Live trading results** — wins, losses, everything
- **Every bug documented** — timezone disasters, API gotchas, 3 AM crashes
